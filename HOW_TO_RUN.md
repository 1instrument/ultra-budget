# How to Run Ultra Budget

This app is stored locally on your computer. Follow these steps to start it anytime.

## 1. Open Terminal
Open your Terminal app (found in Applications > Utilities or search via Spotlight).

## 2. Navigate to the App Folder
Copy and paste this command into the terminal and press Enter:
```bash
cd "/Users/rockwhittington/Desktop/Ultra_Budget"
```

## 3. Run the App
Paste this command and press Enter:
```bash
npm run dev -- --port 3000
```

## 4. Open in Browser
Once the terminal says "Ready", open your browser (Chrome, Safari, etc.) and go to:
**http://localhost:3000**

---

### Important Notes:
- **Keep the Terminal Open:** The app only works while that terminal window is running the command. If you close the terminal, the app stops.
- **Your Data:** All numbers you enter are saved automatically in your browser's local storage.
- **Next Time:** You just need to repeat these steps to start back up.
